# 🚀 Efficient_Page_Replacement_Algorithm_in_Operating_System_2026
Interactive web-based Page Replacement Simulator implementing FIFO, LRU, and Optimal algorithms with real-time visualization, step-by-step execution, and performance metrics like hits and faults.

## 📌 Features

* FIFO (First In First Out)
* LRU (Least Recently Used)
* Optimal Algorithm
* Step-by-step simulation
* Live visualization of frames and faults

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
